凯撒加密解密使用指南:

1.下载项目源文件和CMakeLists.txt至项目文件

2.用终端进入项目文件

3. 进行源码外构建

命令：mkdir build

命令：cd build

命令：cmake ..

4.编译项目

命令：make

生成可执行文件“caesar”

5.运行可执行文件

命令：./caesar
